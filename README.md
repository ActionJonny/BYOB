# BYOB

## npm install

## The applications endpoints are:

### GET requests

* /api/v1/players

* /api/v1/teams

* /api/v1/teams/:id/players

* /api/v1/teams/:id

* /api/v1/players/:id

### POST requests

* /api/v1/players/:id

* /api/v1/teams

* /api/v1/players

### PATCH requests

* /api/v1/teams/:id

* '/api/v1/players/:id

### DELETE requests

* /api/v1/players/:id

* /api/v1/teams/:id
